\section{Общие сведения об универсальных классах}

Пусть $\L$ -- язык, $\K$ -- класс $\L$-систем. Класс $\K$ называется универсальным классом, если он аксиоматизируем и это можно сделать с помощью универсальных формул языка $\L$.

\begin{proposition}\label{prop:AxiomClass}
Класс $\K$ аксиоматизируем тогда и только тогда, когда:
\begin{enumerate}
\item Если $\mathcal{M} \in \K$ и $\mathcal{N} \equiv \mathcal{M}$ тогда $\mathcal{N} \in \K$;
\item Если $\mathcal{M}_i \in \K,$ $i \in I$, $D$ -- ультрафильтр над множеством $I$. Пусть $\overline{\mathcal{M}}$ -- ультра произведение $\mathcal{M}_i$, тогда $\overline{\mathcal{M}} \in K$.
\end{enumerate}
\end{proposition}

Следующее предложение представляет известный факт из теории моделей, доказательство можно найти в \cite{DM} (Теорема 2.3.9).
\begin{proposition}\label{prop:AxiomClass2}
Класс $\K$ универсально аксиоматизируем тогда и только тогда, когда выполнены следующих два условия:
\begin{enumerate}
\item Класс $\K$ аксиоматизируем;
\item Класс $\K$ замкнут относительно операции взятия $\L$-подсистем.
\end{enumerate}
\end{proposition}

\begin{proposition}
Пусть $\K$ -- универсально аксиоматизируемый класс абелевых групп, $p$ -- простое число и для любого $n \in \N$ группа $C(p^n) \in \K$. Тогда:
\begin{enumerate}
\item $\Z \in \K$;
\item $C(p^\infty) \in \K$.  
\end{enumerate}
\end{proposition}

{\bf Доказательство.} 
Пусть $F$ -- фильтр Фреше над множеством натуральных чисел. Рассмотрим фильтрованное произведение групп $C(p^k)$ по фильтру $F$ и обозначим его $A$:
$$A = \prod_{k \in \N} C(p^k) \Big/ F.$$ 
По предложению \ref{prop:AxiomClass} группа $A$ принадлежит классу $\K$. 
Пусть каждая группа $C(p^k)$ порождается элементом $a_i$, тогда, если взять элемент $a = (a_1, a_2, \ldots) \in A$, то он порождает группу $\Z$. Таким образом, группа $\Z$ является подгруппой группы $A$, следовательно по предложению \ref{prop:AxiomClass} группа $\Z \in \K$.

Далее покажем, что группа $C(p^\infty)$ является подгруппой в группе $A$. Для этого, отметим, что группа $C(p^\infty)$ имеет следующее представление:
$$C(p^\infty) = \langle g_1, g_2, g_3, \ldots | \ pg_1 = 1, pg_2 = g_1, pg_3 = g_2, pg_4 = g_3, \ldots \rangle.$$
Выберем произвольный элемент $a \in A$, пусть $a$ имеет вид $a = (a_1, a_2, a_3, \ldots )$. Возьмем в качестве $b_1, b_2, b_3, \ldots$ следующие элементы группы $A$:
$$\begin{array}{ccl}
b_1 & = & (a_1, pa_2, p^2a_3, p^3a_4, \ldots), \\
b_2 & = & (a_1, a_2, pa_3, p^2a_4, \ldots), \\
b_3 & = & (a_1, a_2, a_3, pa_4, \ldots), \\
&  & \ldots \\
\end{array}$$
Заметим, что $pb_k = b_{k-1}$ для всех $k$, следовательно, указанные элементы $b_1, b_2, b_3, \ldots$ порождают группу $C(p^\infty)$, следовательно $C(p^\infty) < A$, и по предложению \ref{prop:AxiomClass2} группа $C(p^\infty) \in \K$. $\square$ 



\section{Постановка задачи}

Пусть $\A$ -- класс абелевых групп в сигнатуре $\L = \{+, -, 0\}.$

\noindent
{\bf Проблема:} {\it Дать классификацию всех универсальных классов из $\A$.}

Пусть $\P$ -- множество простых чисел, а $\P^* = \P \cup \{0\}.$ Определим подкласс $\A_p$ для $p \in \P^*$ следующим образом:
$$\begin{array}{c}
\A_0 = \{A |\text{ если } A \text{ -- группа без кручения, либо } A = \{0\} \}, \\
\A_p = \{A | \ T(A) = T_p(A) \neq 0\}, \ p \neq 0. \\
\end{array}$$ 

Основные задачи для класса $\A_p$, $p \in \P^*$.

\section{Решение основных проблем для класса $\A_0$}

Для доказательства следующего результат нам потребуется два вспомогательных факта, приведем их здесь:
\marginpar{Написать правильную формулировку леммы}
\begin{lemma}\label{lemma:UnivClass}
Любой универсальный класс порождается своими конечно порожденными объектами.
\end{lemma}

Следующая вспомогательная теорема представляет из себя часть объединяющей теоремы из статьи Э.Ю.~Данияровой, А.Г.~Мясникова и В.Н.~Ремесленникова \cite{DMR1}.
\begin{theorem}\label{th:DMR1}{(\cite{DMR1})}
Пусть $A$ и $B$ -- две алгебраические системы нетеровы по уравнениям. Тогда $ucl(A) = ucl(B)$ тогда и только тогда, когда система $A$ дискриминируется системой $B$ и система $B$ дискриминируется $A$.
\end{theorem}

\begin{theorem}
Любые две нетривиальные абелевы группы без кручения универсально эквивалентны.
\end{theorem}

{\bf Доказательство.} Пусть $\A_0$ -- класс всех абелевых групп без кручения в объединении с нулевой группой. Покажем, что $\A_0$ является универсальным классом.
Рассмотрим множество универсальных формул $Ax$, состоящих их двух подмножеств:
\begin{enumerate}
\item Аксиомы абелевых групп;
\item Множество формул $\{\varphi_n | \ n \in \N\}$, где формула $\varphi_n$ имеет вид $$\varphi_n: \  \forall x \ nx = 0 \rightarrow x = 0.$$
\end{enumerate}
Покажем, что множество $Ax$ является множеством аксиом для класса $\A_0$. Пусть группа $A$ удовлетворяет аксиомам из множества $Ax$, тогда группа $A$ является абелевой и любой элемент $a \in A, a \neq 0$ является элементом бесконечного порядка, следовательно группа $A$ без кручения и $A \in \A_0$.

Очевидно, что все группы из $\A_0$ удовлетворяют формулам из множества $Ax$. Следовательно, $Ax$ -- множество аксиом для $\A_0$, и $\A_0$ является универсальным классом.

По лемме \ref{lemma:UnivClass} следует, что класс $\A_0$ порождается своими конечно порожденными группами, следовательно достаточно показать, что для любого $k \in \N$ $ucl(\Z) = ucl(\Z^k)$. По теореме \ref{th:DMR1}, для совпадения универсальных замыканий, нужно показать, что группа $\Z^k$ дискриминируется группой $\Z$, и группа $\Z$ дискриминируется группой $\Z^k$. То, что группа $\Z$ дискриминируется группой $\Z^k$ очевидно, так как $\Z < \Z^k$. 

Покажем, что группа $\Z^k$ дискриминируется группой $\Z$. Для этого, по определению, нужно показать, что для любого количества нетривиальных элементов $h_1, \ldots, h_n \in \Z^k$ существует гомоморфизм $\varphi : \Z^k \rightarrow \Z$, такой, что образы этих элементов не равны 0. Пусть элементы $h_1, \ldots, h_n$ имеют вид:
$$\begin{array}{c}
 h_1 = (\alpha_{11}, \ldots, \alpha_{1k}), \\
 \ldots \\
 h_n = (\alpha_{n1}, \ldots, \alpha_{nk}),
 \end{array} $$
где $\alpha_{ij} \in \Z$. Рассмотрим множество гомоморфизмов, проиндексированных целочисленными векторами длинны $k$, $\varphi_{m_1,\ldots,m_k} : \Z^k \rightarrow \Z$, действующих следующим образом:
$$\varphi_{m_1,\ldots,m_k} (\alpha_1, \ldots, \alpha_k) = m_1 \alpha_1 + m_2 \alpha_2 + \ldots + m_k \alpha_k.$$
Заметим, что если вектор $\overline{\alpha}$ задает элемент группы $\Z^k$, а вектор $\overline{m}$ задает индекс гомоморфизма, то $\varphi_{\overline{m}}(\overline{a})$ представляет скалярное произведение этих двух векторов $\overline{m} \cdot \overline{a}$. Для двух ненулевых векторов скалярное произведение не равно нулю тогда и только тогда, когда они не перпендикулярны. Рассмотрим вектор $\overline{h_1}$, в пространстве $\Z^k$ ему соответствует перпендикулярная ему гиперплоскость размерности $k-1$, аналогично и остальным векторам будут соответствовать гиперплоскости. Всего таких гиперплоскостей конечное число, а следовательно они не могут покрывать все пространство $\Z^k$, поэтому обязательно найдется вектор $\overline{m}$, который не принадлежит ни одной из этих гиперплоскостей, следовательно и гомоморфизм $\varphi_{\overline{m}}$, который будет переводит все элементы $h_1,\ldots, h_n$ в ненулевые. Следовательно, группа $\Z^k$ дискриминируется группой $\Z$. Таким образом, мы доказали, что $ucl(\Z) = ucl(\Z^k)$. $\square$


\begin{corollary}
Если $A$ -- нетривиальная абелева группа, то $\A_0 = ucl(A),$ в частности $\A_0 = ucl(\Z)$.
\end{corollary}

Пусть $p \neq 0$.

\begin{theorem}
\begin{enumerate}
\item Для любого простого числа $p \in \P$ класс групп $\A_p$ является универсальным классом.
\item Аксиомами класса $\A_p$ будут аксиомы абелевых групп и множество формул $\{\varphi_n | \ (n,p)=1\}$, где формула $\varphi_n$ имеет вид:
$$\varphi_n: \forall x \ nx = 0 \rightarrow x = 0.$$
\end{enumerate}
\end{theorem}

Далее пусть $\K$ -- универсальный класс из $\A$ и $p \in \P^*$. Тогда $\K_p = \K \cap \A_p$ -- также универсальный класс и $\K = \bigcup\limits_{p \in \P^*} \K_p.$

Это разложение позволит свести проблему описания универсальных классов из $\A$ к соответствующей проблеме для класса $\A_p$ при фиксированном $p$.

\section{Примарное разложение универсальных классов}

Пусть $\U$ -- универсальный класс абелевых групп и $A \in \U$, $T(A) = \oplus T_p(A)$. Тогда для любого простого числа $p$ группа $T_p(A) \in \U.$ Пусть $\U_p = \U \cap \A_p$ -- будет $p$-примарной компонентой $\U$. Ясно, что $\U_p$ есть также универсальный класс.

Ниже мы определим операцию $*$ $p$-примарное разложение универсальных классов $\U = * \U_p$ и инварианты $I(\U)$. Определим $p$-инвариант $I_p(\U)$ таким образом, что 
$$I(\U) = \bigcup_{p \in \P} I_p(\U),$$
$$Ax(\U) = \bigcup_{p \in \P} Ax(\U_p).$$

Пусть $p \neq 0$ и $\U$ -- универсальный класс абелевых групп такой, что $\U \subset \A_p$. Определим универсальный $p$-инвариант для такого $\U$.

\subsection{Ограниченный случай}

Пусть $\U$ -- ограниченный класс. Тогда существует такое натуральное число $m$, что $p^m T_p(A) = 0$ для всех $A$ из $\U$.

Существует конечная группа $F$ такая, что $\U = ucl(F)$. Тогда $\U = Sub(F)$ (это следует из общих теоретико-модельных теорем).

$$I(\U) = \{F | \text{ где } F \text{ -- конечная группа} \}.$$

Введем характеристику $\chi(\U) = (\alpha_1, \ldots, \alpha_{m-1}, \alpha_m),$ где $\alpha_i \in \N^*$ и $\alpha_j \geq \alpha_{j+1}$, $j=1,\ldots,m-1$.

$$A = C^{k_1}(p) \oplus C^{k_2}(p^2) \oplus \ldots \oplus C^{k_m}(p^m),$$
$\alpha_i = \max (k_i)$ по всем $A$ из $\U$.

\begin{enumerate}
\item Два универсальных класса $\U_1$ и $\U_2$ из $\A_{p^m}$ совпадают тогда и только тогда, когда $\chi(\U_1) = \chi(\U_2).$
\item Для любой допустимой характеристики $\chi$ существует универсальный класс $\U$ такой, что $\chi(\U) = \chi$.
\item Аксиомами $\U$ будет
\end{enumerate}
\end{theorem}

\textbf{Доказательство. }
